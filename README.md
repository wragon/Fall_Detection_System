## 개요
노인 비율이 기하급수적으로 증가하는 우리나라에서 낙상은 노년 생활의 가장 큰 위험 요소입니다.  
치매환자, 독거노인, 기저질환 등 일상 속에서 낙상의 위험이 있는 사람들의 갑작스런 사고에 수월하게 대처할 수 있고  
넘어지고 나서 바로 신고가 이루어 진다면 골든 타임을 놓치는 상황을 막을 수 있습니다.
